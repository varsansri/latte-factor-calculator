# Latte Factor True Cost Calculator
> That $5 coffee isn't costing you $5. Over 20 years, it's costing you $60,000 in lost wealth.

**[Live Demo →](https://varsansri.github.io/latte-factor-calculator)**

## What It Does
Enter a daily habit (name + cost), how often you do it, and how many years. The tool calculates the raw total spent AND the compound opportunity cost (what you'd have if that money was invested instead). Shows real equivalents: months of rent, cars, flights. The final number consistently shocks users into sharing it.
